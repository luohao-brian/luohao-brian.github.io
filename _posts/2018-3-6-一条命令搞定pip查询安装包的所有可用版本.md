### 背景

在使用pip安装python包的时候，如果不指定任何参数，那么会自动安装这个包的最新版本，我们可以用如下的命令来安装一个python包指定的版本，但是怎么才能知道这个包到底有哪些可用的版本呢？

```sh
pip install peewee==3.0.1
```

### 解决办法

首先，要保证pip是9.0以上版本，如果不是，请使用如下命令升级:

```
pip install --upgrade pip
```
