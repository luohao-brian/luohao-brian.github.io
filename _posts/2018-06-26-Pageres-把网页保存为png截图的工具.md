### [](#Pageres)Pageres

Pageres是一个基于nodejs的工具，可以将指定的web页面截获为图像保存。使用npm安装非常简单：

```sh
npm install --global pageres-cli
```

### [](#Demo)Demo

```sh
pageres ygjs.tech
```
